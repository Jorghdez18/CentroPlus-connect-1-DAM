# CentroPlus-Connect

## 📖 Descripción del proyecto

**CentroPlus-Connect** es una aplicación desarrollada en **JavaFX** como proyecto académico de **1º Desarrollo de Aplicaciones Multiplataforma (DAM)**.

La aplicación permite gestionar las actividades de un centro de manera sencilla, ofreciendo una interfaz gráfica intuitiva para consultar actividades disponibles, realizar reservas y visualizar incidencias registradas en el sistema.

El objetivo principal del proyecto es aplicar los conocimientos adquiridos sobre programación orientada a objetos, interfaces gráficas con JavaFX y gestión de datos dentro de una aplicación de escritorio.

---

## 🚀 Funcionalidades principales

### 🏠 Inicio

La pantalla principal muestra un resumen general del sistema, incluyendo:

- Número total de actividades disponibles.
- Número de reservas realizadas.
- Número de incidencias registradas.
- Acceso rápido a las diferentes secciones de la aplicación.

### ⚽ Gestión de actividades

Desde esta sección el usuario puede:

- Consultar las actividades disponibles.
- Visualizar el número de plazas libres de cada actividad.
- Comprobar la disponibilidad actualizada.
- Gestionar las reservas realizadas.

Actividades incluidas en la demostración:

- Fútbol
- Baloncesto
- Programación

### 📅 Gestión de reservas

La aplicación permite registrar reservas de usuarios para las distintas actividades.

Para cada reserva se muestra información como:

- Identificador de la reserva.
- Usuario asociado.
- Actividad reservada.

Además, al realizar una reserva se actualiza automáticamente el número de plazas disponibles.

### 🛠️ Gestión de incidencias

La aplicación incorpora un apartado destinado al control de incidencias donde:

- Se pueden consultar incidencias registradas.
- Se muestra un mensaje cuando no existen incidencias activas.
- Se facilita el seguimiento del estado del sistema.

## 🎓 Objetivos académicos

Con este proyecto se han trabajado los siguientes conceptos:

- Creación de interfaces gráficas con JavaFX.
- Programación Orientada a Objetos (POO).
- Aplicación del patrón Modelo-Vista-Controlador (MVC).
- Gestión de colecciones de datos.
- Navegación entre pantallas.
- Manipulación de eventos y controles gráficos.
